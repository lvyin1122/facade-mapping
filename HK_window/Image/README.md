# Radiant Motion Graph - Art Coding Project

An interactive, animated art piece that creates a beautiful radial burst of light and color with dynamic motion and blur effects.

## Description

This project generates an abstract, vibrant composition featuring:
- A bright, radiant center with intense glow
- Radial rays of color emanating outward
- Rich color palette: fuchsia, magenta, reds, oranges, blues, and golds
- Heavy blur effects creating a dreamlike, ethereal quality
- Dynamic, pulsing motion

## Features

- **Interactive Controls**: Adjust blur intensity, color intensity, motion speed, and ray count in real-time
- **Smooth Animation**: Continuous motion with pulsing and rotation effects
- **Responsive Design**: Automatically adjusts to window size
- **Color Blending**: Smooth transitions between vibrant colors

## How to Run

1. Simply open `index.html` in a web browser
2. No installation or build process required - uses p5.js via CDN

## Controls

- **Blur Intensity**: Controls the amount of blur effect (1-20)
- **Color Intensity**: Adjusts the brightness and saturation of colors (50-150)
- **Motion Speed**: Changes the animation speed (0-3)
- **Ray Count**: Number of radial rays (20-100)
- **Reset Button**: Restarts the animation

## Technologies

- HTML5
- JavaScript
- p5.js (Creative Coding Library)

## Browser Compatibility

Works in all modern browsers that support HTML5 Canvas and ES6 JavaScript.
